Questo è un semplice sistema di prenotazione alberghiera realizzato utilizzando C#, ASP.NET Core MVC e MySQL.

In questo progetto gli utenti possono:

Creare un account, effettuare il login e il logout

Visualizzare le camere disponibili con dettagli come prezzo e tipo di letto

Prenotare o cancellare le proprie prenotazioni in base a specifici intervalli di date

Consultare le proprie prenotazioni in un pannello personale

Un account amministratore (admin@hotel.com, 12345admin) può aggiungere o gestire le camere disponibili per la prenotazione.

Il progetto è completamente integrato con un database MySQL per la gestione dei dati e utilizza Bootstrap 5 per un’interfaccia utente moderna, reattiva e graficamente accattivante.



English: 

This is a simple Hotel Booking System built using C#, ASP.NET Core MVC, and MySQL.

In this project, users can:

Create an account, log in, and log out

Browse available rooms with details such as price and bed type

Reserve or cancel their bookings based on specific date ranges

View their own bookings in a personal dashboard

An Admin account (admin@hotel.com, 12345admin) can add or manage rooms that users can book.

The project is fully integrated with a MySQL database for data storage and uses Bootstrap 5 for a clean, responsive, and visually appealing user interface.
